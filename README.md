# **[TangledKai]_Son of Thieves: Chronicles of Etheriel_v0.8**

A fantasy tale of a descendant of the Thief King on the continent of Etheriel
Enhanced with a platform designed for immersive reading experiences

Here is a complete list of all current features:

* Reading Interface:
  * Immersive Reading Experience: Minimalist, clean, distraction-free interface with focus mode
  * Chapter Navigation: Intuitive chapter browsing with sidebar table of contents
  * Progress Tracking: Visual reading progress indicators with automatic persistent storage
  * Responsive Design: Optimized for desktop, tablet, and mobile devices

* Theme Colors:
  * Dark Night Nest, Parchment Scroll, Pure White Paper, Midnight Shadow, Forest Border
  * Low-key Dark, Amber Gold, Mocha Coffee, Terminal Classic, Abyss Sea

* Typography Control:
  * Font size (10px-32px)
  * Line height customization (0.8-2.5)
  * 4 font options: Noto Serif TC, Noto Sans TC, Noto Sans Mono, Cubic 11

* Music Player:
  * Music note icon in the top-right navigation bar — click to open the player panel
  * Supports play / pause, previous / next track, shuffle mode, and mute toggle
  * Volume slider for real-time adjustment (0–100%)
  * Playlist displays all tracks — click any track to jump directly, currently playing track is highlighted
  * Automatically plays the next track when the current one ends

* Keyboard Shortcuts:
  * ← →: Previous/Next chapter
  * F: Toggle focus mode
  * ESC: Close panels


# **[TangledKai]_盜賊之子：艾瑟瑞爾戰記_v0.8**

講述艾瑟瑞爾大陸上的一名盜賊之王後裔的奇幻故事
並增加為沉浸式閱讀體驗而設計的平台

以下是目前所具備的所有功能列表：

* 閱讀介面：
  * 沉浸式閱讀體驗：簡約、乾淨、無干擾介面，搭配專注模式隱藏所有 UI 元素
  * 章節導覽：直觀的章節瀏覽與側邊欄目錄，支援雷文篇與番外篇分組顯示
  * 進度追蹤：頂部視覺化閱讀進度條與側欄章節完成率，自動持久儲存閱讀位置
  * 響應式設計：針對桌機、平板與手機最佳化

* 主題配色：
  * 暗夜巢穴、羊皮古卷、純白紙張、玄色幽影、邊境森林
  * 低調暗色、琥珀金磚、摩卡咖啡、鄉民本色、深淵之海

* 字型控制：
  * 字體大小：10px–32px 自由調整
  * 行距自訂：0.8–2.5 自由調整
  * 提供4種字體：思源宋體、思源黑體、等寬黑體、像素字體

* 音樂撥放器：
  * 右上角導覽列音符圖示，點擊開啟播放面板
  * 支援播放／暫停、上一首／下一首、隨機播放、靜音切換
  * 音量滑桿即時調整（0–100%）
  * 播放清單列出所有曲目，點擊可直接跳曲，當前播放曲目高亮顯示
  * 播放結束後自動接續下一首

* 快捷鍵：  
  * ← →：上一章/下一章
  * F：切換專注模式
  * ESC：關閉面板
